Простейшайя программа добавления и удаления приказов, сотрудников, должностей и расстановки
Работает с SQLite
1. Запустить файл CreateTables.py для создания БД и таблиц (Пути прописаны в path во всех файлах py)
2. Запустить файл InsertData.py для наполнения таблиц тестовыми данными
3. Для работы запустить TestGUI.py и экспериментировать
